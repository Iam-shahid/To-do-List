# 📝 To-Do List App

A simple **To-Do List** web app built using **HTML, CSS, and JavaScript**.  
You can add tasks, mark them as completed, and delete them easily.  

---

## 🚀 Features
- ➕ Add new tasks
- ✅ Mark tasks as completed (click to toggle)
- ❌ Delete tasks (× button)
- 📱 Responsive & clean UI

---
